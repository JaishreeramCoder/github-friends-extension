Privacy Policy for GitHub Friends

Effective Date: March 2026

Overview

GitHub Friends is a Chrome extension that allows users to privately star GitHub profiles.

Information Collected

The extension stores the following information locally on the user's device using chrome.storage.local:

  • GitHub profile URL
  • Full name
  • Nickname
  • Pronouns
  • Latest public commit time (retrieved from GitHub API)

How Information Is Used

This information is used solely to display starred users within the extension popup.

Data Storage

All data is stored locally on the user's device. The developer does not collect, store, or transmit this data to any external server.

Data Sharing

No user data is sold, shared, or transferred to third parties.

Permissions

  • https://github.com/* — Required to inject the star button on GitHub profile pages.
  • https://api.github.com/* — Required to retrieve publicly available repository data.
  • storage — Required to store starred users locally.
